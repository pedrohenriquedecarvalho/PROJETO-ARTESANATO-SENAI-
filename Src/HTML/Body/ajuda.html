Central de Ajuda

Compras

  • Sou menor de idade, posso realizar compras?

    • - Sim, apenas utilize os dados de um responsável.

  • Consigo acompanhar meus pedidos?

    • - Sim, assim que o pagamento é confirmado, automaticamente, nos seus pedidos, aparece a rota do seu pedido.

  • Caso meu pedido não for entregue, posso pedir reembolso?

    • - Sim, comunique-nos e iremos resolver este problema!

  • Posso cancelar pedidos?

    • - Sim, justifique o cancelamento entrando em contato com a loja para que a ação seja feita.

  • Meu produto veio danificado, há trocas?

    • - Sim, justifique e envie fotografias sobre o dano e podemos trocar seu produto.

Cadastros e Dados

  • Posso atualizar meu cadastro?

    • - Claro! Entre no seu perfil e atualize o que deseja.

  • Em casos de exclusão da minha conta, para onde vai meus dados?

    • - Tratamos todos seu dados com cautela. Em casos de exclusão de sua conta, seus dados serão excluidos e não teremos mais acesso a eles.

  • Errei algum dado no cadastro, posso modificá-lo?

    • - Sim, será a mesma situação de atualização de dados do seu perfil.

  • É necessário criar um cadastro para realizar uma compra?

    • - Você pode comprar como visitante, sem necessidade de criar um cadastro. No entanto, ao criar uma conta, você pode aproveitar melhores benefícios.

Pagamento

  • Como funciona pagamentos via Pix?

    • - Ao selecionar esta forma de pagamento, será gerado uma chave copia e cola para realizar o pagamento via Pix.

  • Posso alterar a forma de pagamento?

    • - Tome cuidado! Esta ação não pode ser alterada.

  • O pagamento é seguro no site? Como posso garantir que meus dados estão protegidos?

    • - Sim, o pagamento no nosso site é totalmente seguro. Utilizamos criptografia SSL (Secure Socket Layer),garantindo a segurança de seus dados pessoais e de pagamentos.

  • O que acontece se eu não conseguir concluir o pagamento? Vou perder meu pedido?

    • - Se você não conseguir concluir o pagamento, o pedido não será finalizado e, portanto, não será processado. No entanto, não se preocupe, o item não será perdido.
